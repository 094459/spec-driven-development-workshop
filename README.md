# Welcome

Welcome to the spec driven development hands on workshop. The goal is to help you understand the key concepts of spec driven development, and provide a hands on workshop that will get you immersed in the workflow.

This workshop can either be run in a traditional workshop environment that is led by a facilitator, or you can work through the exercises in a self service perspective. If you want a deep dive, try and go to a facilitator led session as they will be able to dive deep into the various topics introduced in this workshop, and share good practices and tips to help you write better specs.


## What are you going to need?

- [Kiro](https://kiro.dev/downloads) - you will need to install Kiro, a new IDE that has native support for spec driven development (available for Windows, MacOS, and Linux)
- Python 3.10 (or later)
- [Python uv](https://github.com/astral-sh/uv?tab=readme-ov-file#installation)
- git
